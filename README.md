## 🚀 Sobre o Projeto

Este projeto foi desenvolvido utilizando **Streamlit** para criação de aplicações interativas de dados, permitindo transformar análises em interfaces visuais acessíveis.

O desenvolvimento foi realizado em formato de **co-working com Gabrielanalytics**, focando na construção de soluções práticas e aplicáveis no mundo real.

O Streamlit é amplamente utilizado para criar aplicações de dados com poucas linhas de código, facilitando a visualização e interação com informações complexas. :contentReference[oaicite:0]{index=0}

---

## 🧠 Objetivo

O objetivo deste projeto é:

✔️ Transformar dados em dashboards interativos  
✔️ Facilitar a análise e interpretação de informações  
✔️ Criar aplicações de dados com foco em usabilidade  
✔️ Aplicar conceitos reais de Data Science  

---

## ⚙️ Funcionalidades

- 📊 Visualização de dados interativa  
- 🔍 Filtros dinâmicos  
- 📈 Gráficos e análises em tempo real  
- 💻 Interface amigável via navegador  

---
