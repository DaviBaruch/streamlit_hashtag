🌟 FUNCIONALIDADES PRINCIPAIS

1. ANÁLISE INDIVIDUAL 📈
   • Gráficos interativos (Linha e Candlestick)
   • Indicadores: SMA, Bollinger Bands
   • Métricas: Preço, Volatilidade, Sharpe Ratio, Max Drawdown
   • Download de dados em CSV
   • Suporte a: Ações BR, Ações US, Criptomoedas, Índices

2. COMPARAÇÃO DE ATIVOS ⚖️
   • Comparação de preços normalizados (base 100)
   • Análise de retorno cumulativo
   • Matriz de correlação (heatmap interativo)
   • Até 10 ativos simultâneos
   • Benchmark com índices

3. SIMULADOR DE PORTFÓLIO 💼
   • Alocação customizável
   • Cálculo de retorno ponderado
   • Visualização de crescimento
   • Análise de contribuição por ativo
   • Exportação de simulações

4. INDICADORES TÉCNICOS 📊
   • SMA (20, 50, 200 dias)
   • Bollinger Bands (σ=2)
   • RSI (período 14)
   • MACD (12, 26, 9)
   • Sinais de compra/venda automáticos

🚀 COMO USAR

1. Instalar:
   $ pip install -r requirements.txt

2. Executar:
   $ streamlit run main.py

3. Acessar:
   http://localhost:8501

4. Explorar:
   • Menu lateral para selecionar seção
   • Sidebars com opções de configuração
   • Gráficos interativos
   • Botões de download/exportação
